# 3.1 — Speech-to-Text with Whisper API

## Goal

Add a `transcribe_audio` tool that accepts a local audio file path, calls the OpenAI Whisper API, returns the transcript text plus word-level timestamps, and traces the call with LangSmith's `@traceable` decorator.

---

## Step 1 — Add dependency

In `pyproject.toml` under `dependencies`, add:

```toml
"langsmith>=0.2",
```

Then run:

```bash
uv sync
```

> `openai` is already pulled in transitively via `langchain-openai`, so no extra package is needed for Whisper.

---

## Step 2 — Create the `audios/` folder

```bash
mkdir -p audios
```

Place one or more `.mp3` / `.wav` / `.m4a` sample files here for testing.  
The folder is intentionally kept out of source control — add it to `.gitignore`:

```
audios/
```

---

## Step 3 — Update `.env.example`

```env
# LangSmith (already present — verify these are set)
LANGSMITH_TRACING=true
LANGSMITH_ENDPOINT=https://api.smith.langchain.com
LANGSMITH_API_KEY=your_langsmith_api_key_here
LANGSMITH_PROJECT="your_langsmith_project_name_here"
```

Copy to your `.env` and fill in real values. No new keys are needed; Whisper uses `OPENAI_API_KEY`.

## Step 4 — Create `app/tools/transcription_tool.py`

```python
# app/tools/transcription_tool.py
import os
from pathlib import Path
from openai import OpenAI
from langsmith import traceable
from dotenv import load_dotenv

load_dotenv()

_client = OpenAI()


@traceable(name="transcribe_audio")
def transcribe_audio(file_path: str, language: str = "en") -> dict:
    """
    Transcribe a local audio file using the OpenAI Whisper API.
    Returns the transcript text and word-level timestamps.

    Args:
        file_path (str): Relative or absolute path to the audio file inside audios/.
        language (str): BCP-47 language code. Defaults to 'en'.
    """
    resolved = Path(file_path).resolve()
    if not resolved.exists():
        raise FileNotFoundError(f"Audio file not found: {resolved}")

    print(f"TOOL: Transcribing {resolved.name} ...")

    with open(resolved, "rb") as audio_file:
        response = _client.audio.transcriptions.create(
            model="whisper-1",
            file=audio_file,
            language=language,
            response_format="verbose_json",   # enables word-level timestamps
            timestamp_granularities=["word"],
        )

    words = []
    if hasattr(response, "words") and response.words:
        words = [
            {
                "word": w.word,
                "start": round(w.start, 3),
                "end": round(w.end, 3),
            }
            for w in response.words
        ]

    result = {
        "file": resolved.name,
        "language": response.language,
        "duration_seconds": round(response.duration, 2),
        "transcript": response.text,
        "words": words,
    }

    print(f"TOOL: Transcription complete — {len(words)} words, {result['duration_seconds']}s")
    return result
```

### Key points

| Parameter                          | Why                                                                                                |
| ---------------------------------- | -------------------------------------------------------------------------------------------------- |
| `response_format="verbose_json"`   | Enables metadata (duration, language, segments).                                                   |
| `timestamp_granularities=["word"]` | Populates `response.words` with per-word timing.                                                   |
| `@traceable`                       | Wraps the function in a LangSmith span; the span name matches the function name used in the agent. |

## Step 5 — Verify the tool standalone

Create a quick smoke-test script at the project root (do **not** commit it):

```python
# test_whisper.py  (run once, then delete)
from app.tools.transcription_tool import transcribe_audio

result = transcribe_audio("audios/sample.mp3")
print("Transcript:", result["transcript"])
print("First 5 words:", result["words"][:5])
```

Run it:

```bash
uv run python test_whisper.py
```

Expected output:

```
TOOL: Transcribing sample.mp3 ...
TOOL: Transcription complete — 42 words, 8.5s
Transcript: Hello, welcome to the pronunciation assessment ...
First 5 words: [{'word': 'Hello', 'start': 0.0, 'end': 0.48}, ...]
```

Open LangSmith → your project → you should see a `transcribe_audio` trace with inputs and outputs logged.
