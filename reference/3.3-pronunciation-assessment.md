# 3.3 — Pronunciation Assessment Logic

## Goal

Build a `score_pronunciation` tool that compares a **reference text** (the expected answer) against a **user transcription** (from Whisper), computes word-level accuracy, identifies mispronounced words, and returns a 0–100 score — all traced in LangSmith.

---

## Scoring Algorithm Overview

```
Reference text  →  normalise  →  ref_words[]
User transcript →  normalise  →  user_words[]
                                        │
                            Levenshtein alignment
                                        │
                    ┌───────────────────┴───────────────────────┐
                    │                                           │
              correct_words[]                        mispronounced_words[]
                    │
          score = (correct / total_ref_words) × 100
```

Normalisation: lowercase, strip punctuation — so "Hello!" == "hello".

## Step 2 — Create `app/tools/pronunciation_tool.py`

```python
# app/tools/pronunciation_tool.py
import re
from langsmith import traceable
from rapidfuzz import fuzz

# --- helpers -----------------------------------------------------------------

def _normalise(text: str) -> list[str]:
    """Lowercase and strip punctuation, return word list."""
    text = text.lower()
    text = re.sub(r"[^\w\s']", "", text)   # keep apostrophes (contractions)
    return text.split()


def _is_match(ref_word: str, user_word: str, threshold: int = 80) -> bool:
    """Return True when similarity score >= threshold (0-100)."""
    return fuzz.ratio(ref_word, user_word) >= threshold


# --- main scoring function ---------------------------------------------------

@traceable(name="score_pronunciation")
def score_pronunciation(reference_text: str, user_transcript: str) -> dict:
    """
    Compare reference_text against user_transcript and return a pronunciation score.

    Args:
        reference_text (str): The expected sentence the user should have spoken.
        user_transcript (str): The Whisper transcription of what the user actually said.

    Returns:
        dict with keys:
            score (int 0-100),
            correct_words (list[str]),
            mispronounced_words (list[dict]),
            total_ref_words (int),
            matched_words (int)
    """
    ref_words = _normalise(reference_text)
    user_words = _normalise(user_transcript)

    if not ref_words:
        return {
            "score": 0,
            "correct_words": [],
            "mispronounced_words": [],
            "total_ref_words": 0,
            "matched_words": 0,
        }

    correct_words = []
    mispronounced_words = []

    # Walk reference words; for each one find the closest user word
    for ref_word in ref_words:
        best_score = 0
        best_user_word = None

        for user_word in user_words:
            s = fuzz.ratio(ref_word, user_word)
            if s > best_score:
                best_score = s
                best_user_word = user_word

        if _is_match(ref_word, best_user_word or "", threshold=80):
            correct_words.append(ref_word)
        else:
            mispronounced_words.append(
                {
                    "expected": ref_word,
                    "heard": best_user_word or "<missing>",
                    "similarity": best_score,
                }
            )

    matched = len(correct_words)
    total = len(ref_words)
    score = round((matched / total) * 100) if total else 0

    result = {
        "score": score,
        "correct_words": correct_words,
        "mispronounced_words": mispronounced_words,
        "total_ref_words": total,
        "matched_words": matched,
    }

    print(
        f"TOOL: Pronunciation score {score}/100 "
        f"({matched}/{total} words correct, {len(mispronounced_words)} mispronounced)"
    )
    return result
```

## Step 3 — Understanding the similarity threshold

| `threshold`  | Behaviour                                                    |
| ------------ | ------------------------------------------------------------ |
| 100          | Exact match only — penalises minor accent variations harshly |
| 80 (default) | Tolerates 1–2 character edits (e.g. "their" ↔ "there")       |
| 70           | Lenient — good for non-native speakers in early practice     |

Expose this as an env variable if you want tutors to configure strictness:

```env
# .env.example
PRONUNCIATION_THRESHOLD=80
```

## Step 5 — Tracing custom scoring in LangSmith

Because `score_pronunciation` is wrapped with `@traceable`, every call creates a span in LangSmith with:

- **Inputs**: `reference_text`, `user_transcript`
- **Outputs**: the full result dict
- **Latency**: near-instant (no LLM call)

This lets you:

1. Review every assessment in LangSmith alongside the Whisper transcription span.
2. Filter traces by `score < 50` to identify users who need extra help.
3. Export traces to a dataset to fine-tune scoring thresholds over time.

## Step 6 — Generating feedback text (optional LLM enhancement)

If you want human-readable feedback beyond the score, add an LLM step:

```python
from langchain_openai import ChatOpenAI
from langsmith import traceable

_llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.3)

@traceable(name="generate_pronunciation_feedback")
def generate_pronunciation_feedback(score_result: dict) -> str:
    """Generate a friendly, constructive feedback message from a score result."""
    mispronounced = score_result.get("mispronounced_words", [])
    score = score_result.get("score", 0)

    if not mispronounced:
        return "Excellent! Your pronunciation was perfect."

    word_list = ", ".join(
        f"'{m['expected']}' (you said '{m['heard']}')" for m in mispronounced
    )

    prompt = (
        f"A student scored {score}/100 on a pronunciation exercise. "
        f"The following words were mispronounced: {word_list}. "
        "Write 2-3 sentences of friendly, encouraging feedback and tips for improvement."
    )
    response = _llm.invoke(prompt)
    return response.content
```
